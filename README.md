# 1.1 Алгебра и всякие приколы по мелочи

**Эта часть шпоры скорее на всякий случай, большиство этих задач проще решить кодом (кроме пырычей)**

**Теорема Виета**
Для квадратного уравнения $ax^2 + bx + c = 0$ с корнями $x_1$ и $x_2$ работают два главных правила. Сумма корней равна минус второй коэффициент делить на первый. Произведение корней равно свободный член делить на первый.

$x_1 + x_2 = -\frac{b}{a}$
$x_1 \cdot x_2 = \frac{c}{a}$

Важно помнить, что старший коэффициент $a$ не должен быть равен нулю, иначе уравнение ваще не квадратное.

**Часто в задачах просят найти не сами корни, а выражения через них. Ака сумма и так далее. Используй эти правила.**

Сумма обратных величин равна сумме корней делить на произведение корней.
$\frac{1}{x_1} + \frac{1}{x_2} = \frac{x_1 + x_2}{x_1 \cdot x_2}$

Сумма квадратов корней равна квадрат суммы корней минус удвоенное произведение.
$x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2 \cdot x_1 \cdot x_2$

Сумма кубов корней равна куб суммы корней минус утроенное произведение корней на сумму корней.
$x_1^3 + x_2^3 = (x_1 + x_2)^3 - 3 \cdot x_1 \cdot x_2 \cdot (x_1 + x_2)$

**Последняя цифра числа**
Чтобы найти последнюю цифру степени, достаточно смотреть только на последнюю цифру основания. Цифры повторяются циклами длиной 4. Например, для степени двойки последовательность последних цифр: 2, 4, 8, 6, затем снова 2.

Если степень очень большая, вручную считать долго. Используй функцию `pow` в Python, которая сразу возвращает остаток от деления на 10.

```python
last_digit = pow(base, exp, 10)
```

Для суммы нечётных чисел есть простое правило. Сумма первых $n$ нечётных чисел равна $n$ в квадрате.
$1 + 3 + 5 + ... + (2n - 1) = n^2$

Если нужно найти количество слагаемых в ряду $1 + 3 + ... + k$, то $n = \frac{k + 1}{2}$.

**Наибольший общий делитель степеней**
Если видишь выражение вида $\text{НОД}(a^n - 1, a^m - 1)$, используй свойство степеней. НОД таких чисел равен основанию в степени НОД показателей минус единица.

$\text{НОД}(a^n - 1, a^m - 1) = a^{\text{НОД}(n, m)} - 1$

Сначала найди НОД показателей степени, потом возводи основание в эту степень и вычитай 1. В Python это считается так:

```python
import math
n = 27
m = 18
gcd_pow = math.gcd(n, m)
result = 10**gcd_pow - 1
```

**Суммы дробей**
Если видишь сумму вида $\frac{1}{n \cdot (n+1)}$, её можно раскрыть как разность двух дробей.
$\frac{1}{n \cdot (n+1)} = \frac{1}{n} - \frac{1}{n+1}$

При сложении таких слагаемых большинство членов сокращается. Остаются только первое слагаемое первой дроби и второе слагаемое последней дроби.

**Когда писать код**
Если в задаче нужно перебрать значения, посчитать сумму большого ряда или найти остаток от деления большого числа, не делай это вручную. Напиши скрипт. Для уравнений с параметром можно перебрать возможные значения параметра в цикле и проверить условие на сумму корней. Но идея это чаще всего не лучшая. Используй только в неочевидных уравнениях, где иначе не думается.

```python
for a in range(-100, 101):
    if a == 0:
        continue
    sum_roots = 9 * a / a
    if sum_roots == 9:
        print(a)
```

Помни, что для квадратного уравнения старший коэффициент не должен быть равен нулю.


# 1.2 Теория чисел. 
**Большинство таких задач (по типу сколько есть чисел таких, что) так же решлось кодом с перебором, но важные понятия отметить стоит.**

**Последняя цифра числа**
Последняя цифра числа — это остаток от деления этого числа на 10. При сложении или умножении больших чисел можно искать остаток от деления на 10 для каждого слагаемого или множителя отдельно.

Для суммы квадратов натуральных чисел от 1 до $n$ есть готовая формула. Сумма равна произведению $n$, $n+1$ и $2n+1$, делённому на 6.
$1^2 + 2^2 + ... + n^2 = \frac{n(n+1)(2n+1)}{6}$

Для суммы нечётных чисел от 1 до $2n-1$ результат всегда равен квадрату количества слагаемых. Если последнее число в сумме равно $k$, то количество слагаемых равно $\frac{k+1}{2}$.
$1 + 3 + 5 + ... + (2n-1) = n^2$

Если в произведении нечётных чисел есть число, оканчивающееся на 5, то всё произведение будет оканчиваться на 5. Это работает, потому что среди множителей нет чётных чисел, которые могли бы превратить 5 в 0.

В Python можно сразу вычислять огромные степени и брать остаток от деления. Используй функцию `pow` с тремя аргументами.

```python
last_digit = pow(base, exp, 10)
sum_squares = (n * (n + 1) * (2 * n + 1)) // 6
last_digit_sum = sum_squares % 10
```

**Наибольший общий делитель степеней**
Если видишь выражение вида $\text{НОД}(a^n - 1, a^m - 1)$, где $a$ — натуральное число, используй свойство показателей степеней. НОД таких чисел равен основанию степени $a$ в степени НОД показателей $n$ и $m$ минус единица.

$\text{НОД}(a^n - 1, a^m - 1) = a^{\text{НОД}(n, m)} - 1$

Сначала найди наибольший общий делитель показателей степени, затем возvedi основание в эту степень и вычти 1.

В Python можно считать такие числа напрямую, так как язык поддерживает целые числа любой длины. Модуль `math` содержит функцию для поиска НОД.

```python
import math
n = 27
m = 18
a = 10
val1 = a**n - 1
val2 = a**m - 1
gcd_val = math.gcd(val1, val2)
```

**Поиск чисел по условию делимости**
Если нужно найти все натуральные числа $n$, не превосходящие некоторого предела, при которых дробь или выражение делится нацело на заданное число, не пытайся решать сложное уравнение вручную. Проще перебрать все возможные значения в цикле.

Проверяй условие с помощью оператора остатка от деления `%`. Если остаток равен 0, значит деление нацело.

```python
limit = 300
divisor = 143
result_list = []
for n in range(1, limit + 1):
    numerator = n**2 - 16 * n + 63
    if numerator % divisor == 0:
        result_list.append(n)
print(result_list)
```

**Сумма коэффициентов многочлена**
Если в задаче дан многочлен от переменной $x$ в какой-то степени и нужно найти сумму всех его коэффициентов, подставь вместо $x$ единицу. Значение многочлена при $x=1$ и будет равно сумме коэффициентов.

Для выражения вида $(P(x))^k$ сумма коэффициентов равна $(P(1))^k$.

```python
x = 1
polynomial_value = (x**2025 - x**2024 + 1)**2026
sum_coefficients = polynomial_value
```

**Вероятность и делимость**
В задачах на вероятность часто нужно посчитать, сколько чисел на отрезке делятся на заданное число. Количество чисел от 1 до $N$, делящихся на $k$, равно целой части от деления $N$ на $k$.

Если нужно найти числа, которые делятся на $A$ и не делятся на $B$, сначала найди количество делящихся на $A$, затем вычти из них те, которые делятся и на $A$, и на $B$. Числа, делящиеся на оба, делятся на наименьшее общее кратное $A$ и $B$.

```python
import math
N = 500
A = 3
B = 12
lcm_val = (A * B) // math.gcd(A, B)
count_A = N // A
count_both = N // lcm_val
count_result = count_A - count_both
```


# 1.3 Комбинаторика и вероятность
**Задачки кодом решаются куда дольше, чем аналитически, этот раздел очень важно запомнить.**

**Правило умножения**
Если нужно выбрать один объект из первой группы и один объект из второй группы, перемножь количество вариантов в каждой группе. Например, если нужно выбрать одного мальчика из 8 и одну девочку из 9, то всего способов будет 8 умножить на 9.

**Сочетания**
Используй сочетания, когда нужно выбрать группу объектов и порядок внутри группы не важен. Например, при выборе команды дежурных или волейбольной команды. Формула числа сочетаний из $n$ по $k$ равна факториалу $n$, делённому на произведение факториала $k$ и факториала разности $n$ минус $k$.

$C_n^k = \frac{n!}{k! \cdot (n-k)!}$

В Python есть готовая функция для этого, чтобы не считать факториалы вручную.

```python
import math
n = 17
k = 3
ways = math.comb(n, k)
```

Если нужно выбрать несколько групп одновременно, например 2 мальчиков из 8 и 2 девочек из 11, перемножь результаты сочетаний для каждой группы.

```python
boys = 8
girls = 11
ways = math.comb(boys, 2) * math.comb(girls, 2)
```

**Перестановки**
Используй перестановки, когда нужно расставить все объекты в ряд и важен порядок. Например, при составлении слова из букв. Если все буквы разные, число способов равно факториалу количества букв.

$P_n = n!$

Если среди букв есть одинаковые, раздели общий факториал на факториалы количества каждой повторяющейся буквы. Например, в слове из 7 букв, где буква «А» встречается 2 раза, нужно разделить 7! на 2!.

```python
import math
total_letters = 7
repeats = 2
ways = math.factorial(total_letters) // math.factorial(repeats)
```

**Вероятность**
Вероятность события равна отношению количества благоприятных исходов к общему количеству возможных исходов.

$P = \frac{\text{благоприятные}}{\text{все}}$

Для кубиков общее количество исходов всегда 36, так как у каждого из двух кубиков по 6 граней. Если нужно найти вероятность того, что сумма очков делится на число, проще перебрать все варианты в цикле.


```python
total = 0
favorable = 0
for i in range(1, 7):
    for j in range(1, 7):
        total += 1
        if (i + j) % 4 == 0:
            favorable += 1
probability = favorable / total
```

**Распределение одинаковых предметов**
Если нужно распределить $n$ одинаковых предметов между $k$ людьми, и люди могут не получить ничего, используй формулу сочетаний с повторениями. Число способов равно сочетанию из суммы $n$ и $k$ минус 1 по $k$ минус 1.

$C_{n+k-1}^{k-1}$

Например, для 10 яблок и 3 человек это будет сочетания из 12 по 2.

```python
import math
n = 10
k = 3
ways = math.comb(n + k - 1, k - 1)
```

**Ограничения и условия**
Если в задаче есть условие, например слово не может начинаться с определённой буквы, посчитай общее количество вариантов и вычти те, которые нарушают условие. Для слов с повторяющимися буквами сначала зафиксируй запрещённую букву на первом месте, посчитай перестановки остальных, затем вычти из общего числа.

Для задач на строки из нулей и единиц, где запрещены определённые комбинации, например два нуля подряд, удобно писать перебор в Python.

```python
count = 0
for i in range(2**10):
    s = bin(i)[2:].zfill(10)
    if '00' not in s:
        count += 1
```

Этот код проверяет все строки длины 10 и считает те, где нет двух нулей подряд (но если нулей хотя бы 20, то уже мейби и не сработает, довольно тяжелая проверка)

## Работа с файлами

Для чтения текстовых файлов и матриц смежности используй стандартную функцию `open`. Режим `'r'` означает чтение, `'w'` — запись. Всегда закрывай файл или используй конструкцию `with`.

```python
with open('file.txt', 'r') as f:
    content = f.read()
    lines = f.readlines()
```

Если числа в строке разделены пробелами или запятыми, используй метод `split`. Он превращает строку в список строк.

```python
text = "1,2,3,4"
numbers = text.split(',')
```

Для работы с архивами в задачах иногда требуется модуль `zipfile`.

```python
import zipfile
with zipfile.ZipFile('archive.zip', 'r') as z:
    z.extractall()
```

Библиотека `pandas` нужна для работы с таблицами. Обычно её импортируют под именем `pd`.

```python
import pandas as pd
df = pd.read_csv('table.csv')
```

Чтобы узнать размер таблицы, используй атрибут `shape`. Первое число — строки, второе — столбцы.

```python
rows, cols = df.shape
```

## Выборка и фильтрация

Чтобы выбрать только те строки, где выполняется условие, напиши условие внутри квадратных скобок.

```python
filtered = df[df['age'] > 18]
```

Можно комбинировать условия через `&` (и) или `|` (или). Каждое условие обязательно бери в скобки.

```python
filtered = df[(df['age'] > 18) & (df['city'] == 'Moscow')]
```

Чтобы выбрать конкретные столбцы, перечисли их названия в списке.

```python
subset = df[['name', 'age']]
```

## Группировка и агрегация

Если нужно посчитать сумму, среднее или количество для каждой группы, используй `groupby`. Сначала указываем колонку для группировки, потом колонку для расчёта и метод.

```python
result = df.groupby('category')['value'].sum()
```

Для подсчёта количества элементов в каждой группе используй `size` или `count`.

```python
counts = df.groupby('category').size()
```

Чтобы найти количество уникальных значений в колонке, используй `nunique`.

```python
unique_count = df['user_id'].nunique()
```

Для поиска самого частого значения используй `value_counts`.

```python
freq = df['color'].value_counts()
most_common = freq.idxmax()
```

## Объединение таблиц

Если данные разбиты на несколько файлов, их нужно соединить по общему ключу. Используй `merge`. Параметр `on` указывает название колонки, которая есть в обеих таблицах.

```python
merged = df1.merge(df2, on='post_id')
```

Если названия ключей разные, укажи их отдельно.

```python
merged = df1.merge(df2, left_on='user_id', right_on='id')
```

```python
text = "Hello World"
text = text.lower()
```

Метод `count` считает количество вхождений подстроки.

```python
num_a = text.count('a')
```

Метод `split` разбивает строку на список слов по пробелам.

```python
words = text.split()
```

Чтобы убрать лишние пробелы по краям, используй `strip`.

```python
clean = "  text  ".strip()
```
